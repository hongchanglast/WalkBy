随身行(WalkBy)

西电的课程表自动化软件

Download this project as a .zip file Download this project as a tar.gz file 
现在的类似课程表的软件，不管是在PC平台还是在移动平台上都有很多现成的应用，但是普及率依然不高。究其原因，在于没有特化和自动化，试想，如果用户有一个课程表，但是还要根据课程表的内容，根据日期和时间一一导入到课程表软件中，这一过程是多么的繁杂。

而且有的课程表软件是基于周的，也就是说软件展示的只是每周一样的课程，但是用户的课程并不是每一周都一样的，有的课程是奇数周上，有的课程是偶数周上，让用户根据这种情况对所有的课程一一进行编辑是不现实的。

基于上述考虑，随身行的出现有效的解决了用户编辑自己的课程表的繁杂过程，实现一键导入课程表。因为课程表是针对学生使用的，而往往当学生选课之后，会得到一个课程表的电子表格或网页。针对每个学校固定使用的课程表格式进行解析，这样就免去了手动编辑课程表的过程。

当然随身行会成立一个开发者联盟，软件接口与使用方法会公布在`GitHub`上，产品会吸引各个学校的相关兴趣个人或组织进行维护自己学校课程表的解析功能，并授予随身行开发者勋章，对于突出贡献者，随身行会授予突出贡献奖。
